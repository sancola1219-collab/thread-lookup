# 🔩 螺牙管件查詢圖鑑

單檔 HTML 的螺牙／管件牙查詢工具。雙擊 `index.html` 即可使用，免安裝、免網路。

**📱 線上版（手機、電腦直接開）：https://sancola1219-collab.github.io/thread-lookup/**

> 手機加到主畫面：Safari／Chrome 開啟上面網址 → 分享或選單 → 「加入主畫面」，之後像 App 一樣點開。

## 功能

- **五大類 65 筆規格**：PT 錐牙（BSPT/R/Rc）、PF 直牙（BSPP/G）、NPT 美規錐牙、公制 M（粗/細牙）、美制 UNC/UNF
- **近遠看**：SVG 畫布依「真實比例」畫出每個牙的外徑圓 — 滾輪縮放、拖曳平移、遠看比大小、近看讀尺寸
- **所有尺寸都在卡片上**：外徑（mm／吋）、牙數 TPI、牙距、攻牙底孔、錐度
- **說明欄**：點卡片看一般叫法（4分、6分、吋半…）、正式標稱、用途、密封方式；固定分頁有「材質總表」與「怎麼分辨 PT/NPT/PF」小抄
- **搜尋**：打「4分」「1/2」「M10」「npt」，或直接打量到的外徑 mm（如「21」）
- **🧊 3D 檢視**：點卡片 →「3D 檢視」，依真實外徑、牙距、牙角（55°/60°）、錐度生成螺牙 3D 模型，拖曳旋轉、雙指縮放（零依賴，自寫軟體渲染）

## 使用

```
雙擊 index.html（或任何靜態伺服器）
```

| 操作 | 方式 |
|---|---|
| 縮放 | 滑鼠滾輪／＋－按鈕／雙指 |
| 平移 | 拖曳 |
| 回全覽 | ⛶ 按鈕 |
| 開關分類 | 上方 PT/PF/NPT/M/UN 色塊 |

## 資料依據

- BSPT/BSPP：ISO 228 / BS 21 / JIS B0202-0203
- NPT：ASME B1.20.1
- 公制：ISO 261/262（底孔＝外徑−牙距）
- UNC/UNF：ASME B1.1
- 攻牙底孔標「約」者為業界慣用參考值，實際請依材質微調
