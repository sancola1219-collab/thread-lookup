# 螺牙查詢 — AI 接手必讀

> 給 Claude / Codex / 任何模型：接手前依序讀 **本檔 → HANDOFF.md → lessons/**。

## 這是什麼

單檔 HTML 螺牙查詢圖鑑（`index.html`），可縮放的真實比例 SVG 卡片牆＋右側說明欄。
使用者是水電／機械實務背景，介面與文件一律**繁體中文**、用台灣術語（4分、6分、白鐵、生料帶）。

## 鐵則

1. **維持單檔零依賴** — 不引入框架、不拆檔、不加 build step。使用者要雙擊即用。
2. **尺寸資料不可用「大概」** — 改資料要註明依據標準（ISO 228 / ASME B1.20.1…）；底孔參考值一律標「約」。
3. **文件慣例** — 一課一檔（lessons/）、開頭一行摘要、附為什麼、發現錯誤即刪。
4. 改完要驗證：開瀏覽器確認 (a) 初始全覽不是空白 (b) 搜尋「4分」「21」結果正確 (c) 點卡片右欄有內容 (d) 無 console error。
