# 螺牙查詢工具 — 設計文件（2026-07-10）

## 一句話
單檔 HTML 的螺牙／管件牙查詢圖鑑：真實比例 SVG 畫布可縮放平移（近遠看），
每個規格的尺寸全部標在卡片上，右側說明欄含一般叫法、材質、密封方式。

## 需求（來自使用者原話）
- 集合所有（常用）螺牙
- 生成管件查詢的 UI 圖
- 能夠近遠看（縮放）
- 所有尺寸都在上面
- 一般叫法、材質相關資料 → 說明欄
- 要有交接文件讓 Codex / 其他模型接手
- 上 GitHub

## 替使用者做的決定（未經逐題確認，可改）
| 決定 | 理由 |
|---|---|
| 單檔 index.html、零依賴 | 比照使用者「繪圖軟體」專案慣例：雙擊即用、免安裝 |
| 涵蓋 PT、PF、NPT、公制M、UNC/UNF 五大類 | 管件牙三種是台灣水電主流；M 與 UNC/UNF 是「所有螺牙」常見延伸 |
| SVG 真實比例圓 + wheel zoom / drag pan | 「近遠看」最直覺的做法；遠看比大小、近看讀數字 |
| GitHub repo 設 private | 保守預設，隨時可改 public |
| 介面全繁體中文 | 使用者慣用語言 |

## 架構
```
index.html（唯一程式檔）
├─ <style>   版面：上工具列 / 左畫布 / 右說明欄
├─ <svg>     畫布：五個群組（每類一橫排卡片），g#world 承載 pan/zoom transform
└─ <script>
   ├─ DATA        五大類規格陣列（尺寸、叫法、底孔、用途）
   ├─ MATERIALS   材質總表（SUS304/316、黃銅、碳鋼、PVC…）
   ├─ render()    依 DATA 生成卡片（圓依真實比例，SCALE px/mm）
   ├─ pan/zoom    pointer events + wheel（以游標為中心縮放）、雙指縮放
   ├─ search      搜尋框：比對規格名/叫法/外徑數字，命中高亮、其餘變暗
   └─ panel       點卡片 → 右欄顯示詳細；固定分頁：材質總表、辨識小抄
```

## 資料正確性
- BSPT/BSPP 外徑取自 ISO 228 / BS 21（例：1/2" = 20.955mm）
- NPT 取自 ASME B1.20.1（例：1/2" = 0.840" = 21.336mm）
- 公制底孔 = 外徑 − 牙距（標準公式）；PT/NPT 底孔為業界慣用參考值，卡片上標「約」
- 台灣俗稱：1/8"=1分、1/4"=2分、3/8"=3分、1/2"=4分、3/4"=6分、1"=1吋、
  1-1/4"=吋2、1-1/2"=吋半、2"=2吋

## 驗收標準
1. 雙擊 index.html 開啟，初始畫面一眼看到五大類全部卡片（遠看）
2. 滾輪放大後每張卡片的尺寸文字清楚可讀（近看）
3. 搜尋「4分」或「21」能找到 PT 1/2 相關卡片
4. 點卡片右欄出現叫法/用途/密封；材質分頁有完整總表
5. 無 console error
